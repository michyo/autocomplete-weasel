Autocomplete Weasel - Atom autocomplete+ provider
=================================================

Overview
--------

Extencible autocomplete+ provider for Weasel Game Engine.

How to use
----------

Theme could be installed in Atom or with

<code>
  apm install autocomplete-weasel
</code>
